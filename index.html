JSL Autohuolto

WordPress CMS / HTML / CSS

Sain suunnitella sivuston rakenteen täysin itse. Alasivut olivat tiedossa, mutta tuotin sisällöt kaikki itse. Lopputuloksena hienot autokorjaamoaiheiset verkkosivut. Asiakas oli todella tyytyväinen.

Pihanne

WordPress CMS / HTML / CSS

Asiakas tiesi mitä halusi sivustolta, jonka pohjalta tein toteutuksen. Lopputuloksena syntyi raikkaat piha-aiheiset verkkosivustot. Sain asiakkaalta paljon positiivista palautetta.

Ame Product

WordPress / WooCommerce / HTML / CSS / PHP

Valvontalaitteistojen verkkokauppatoteutus. Asiakkaalla oli entuudestaan vanha verkkokauppa, josta tuotteet siirrettiin. Suunnittelin sivuston rakenteen itse, mutta sivuston värimaailmaan hyödynnettiin asiakkaan logoa.

Boutique Francoise

WordPress / WooCommerce / HTML / CSS / PHP

Naisten juhla-asusteiden sekä vapaa-ajan vaatteiden kauppa. Ohjenuorana oli luoda "glitterikäs" naisille kohdistettu verkkokauppa, johon yhdistettiin Odoo ERP:in kassajärjestelmä.

A-Rent-O

WordPress / WooCommerce / HTML / CSS / PHP

Uusi palvelu autokouluille, jossa autokoulu vuokraa oppilaille mopokalustot ja harjoitteluradan. Minulle annettiin melko vapaat kädet suunnitella sivuston rakenne ja visuaaliset linjaukset. Projektiin liitettiin myös erillinen varauskalenteri, jonka kautta mopojen varaukset tehdään.

Shopping List Application

Vanilla JavaScript / HTML / CSS

Kauppalista -applikaatio. Rakennettu responsiivisesti käyttäen JavaScript, HTML & CSS. Toimintoina itemien yliviivaus ja poisto.


Temperature Application

React / HTML / CSS

Reactilla tehty lämpötila-applikaatio, johon syötetään halutun kaupungin nimi jonka jälkeen appi hakee kaupungin lämpötilatiedot openweathermapin rajapinnasta. Applikaatiossa on myös mahdollisuus tallentaa haluttu kaupunki selaimen localstorageen, jolloin sivua avatessa oletuksena löytyy viimeisimmäksi tallennettu kaupunki ja sen lämpötilatiedot. Applikaatio uploadattu herokuun.

JavaScript / MongoDB / HTML / CSS
